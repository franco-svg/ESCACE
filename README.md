# ESCACE

Proyecto Final de la carrera **Analista Programador**  
Universidad Abierta Interamericana (UAI)

## Descripción

ESCACE es una aplicación orientada a la **gestión integral del ciclo de vida de los proyectos**, encuadrada dentro de la categoría de sistemas **Project Portfolio Management (PPM)**.

La solución permite administrar tanto los aspectos **estratégicos y administrativos** (clientes, contratos, presupuestos, fechas de vigencia) como los aspectos **operativos** de cada proyecto, incluyendo la gestión de recursos humanos, la planificación del backlog por proyecto y el seguimiento de objetivos y avances a lo largo del tiempo.

ESCACE busca centralizar en un único entorno la **visión global del portafolio de proyectos** y la **gestión operativa diaria**, brindando control, trazabilidad y orden en todo el proceso.

## Arquitectura

Aplicación desarrollada bajo una arquitectura de 5 capas

- BE (Business Entities)
- BLL (Business Logic Layer)
- MPP (Mapper)
- UI (User Interface)
- SEC (Security)

## Tecnologías

- C#
- .NET
